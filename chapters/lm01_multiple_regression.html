üìå 1. Overview & Purpose

Multiple Linear Regression extends simple regression by using two or more independent variables to explain a dependent variable.

Why Use Multiple Regression?

  • Identify relationships between variables
  • Test existing theories about factor relationships
  • Forecast stock returns, earnings, bond prices, etc.
  • Control for confounding factors by including them as independent variables
Multiple regression isolates the effect of each independent variable while holding others constant ‚Äî this is the key advantage over simple regression.

‚ö†Ô∏è 3. Assumptions of Multiple Regression

For valid inference, the model must satisfy these assumptions:

  1. Linearity: Linear relationship between Y and each X
  2. Homoskedasticity: Constant variance of residuals across all observations
  3. No Serial Correlation: Errors are independent of each other (no autocorrelation)
  4. Normality: Residuals are normally distributed
  5. No Multicollinearity: No exact linear relationship among independent variables
  6. No Endogeneity: Errors are uncorrelated with independent variables
Violations of assumptions don't invalidate the model ‚Äî they affect the reliability of statistical tests and inferences.

üìä Example: Stock Return Model

Model: Return = 0.02 + 0.8(Market) + 0.3(Size) + 0.1(Value)

  • Market Œ≤ (0.8): 1% increase in market ‚Üí 0.8% increase in return (holding size and value constant)
  • Size Œ≤ (0.3): Effect of firm size factor
  • Value Œ≤ (0.1): Effect of value factor

üéØ 7. Prediction

Predicted Value ≈∂ = b0 + b1X1 + b2X2 + ... + bkXk
üìä Example: Earnings Forecast

Model: EPS = 0.50 + 0.03(Revenue) + 0.02(Assets)

If Revenue = 100 and Assets = 50:

EPS = 0.50 + 0.03(100) + 0.02(50) = 0.50 + 3 + 1 = 4.50

Prediction intervals are wider than confidence intervals because they account for both estimation error AND inherent variability in Y.
