üìê LM 1: Multiple Regression

Quantitative Methods ‚Ä¢ CFA Level II

üìå Overview

Multiple Linear Regression extends simple regression by using two or more independent variables to explain a dependent variable.

Why Use It?

  • Identify relationships between variables
  • Test theories about factor relationships
  • Forecast returns, earnings, prices
  • Control for confounding factors
Multiple regression isolates each variable's effect while holding others constant ‚Äî the key advantage over simple regression.

Types of Investment Problems

  • Stock return prediction (factor models)
  • Earnings forecasting
  • Bond yield analysis
  • Economic factor analysis

üìê The Regression Model

General Form
$$Y_i = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_k X_k + \varepsilon_i$$
Term Meaning
$Y_i$ Dependent variable
$\beta_0$ Intercept (Y when all X = 0)
$\beta_k$ Partial slope for $X_k$
$\varepsilon_i$ Error term (residual)
$k$ Number of independent variables
Partial slope interpretation: $\beta_1$ = expected change in Y for a 1-unit change in $X_1$, holding all other X constant.

Example

üìä Stock Return Model

Return = 0.02 + 0.8(Market) + 0.3(Size) + 0.1(Value)

  • 0.02: Expected return when all factors = 0
  • 0.8: 1% ‚Üë market ‚Üí 0.8% ‚Üë return
  • 0.3: Size factor effect

‚ö†Ô∏è Assumptions

For valid inference, the model must satisfy:

  1. Linearity: Linear relationship between Y and each X
  2. Homoskedasticity: Constant variance of residuals
  3. No Serial Correlation: Errors are independent
  4. Normality: Residuals are normally distributed
  5. No Multicollinearity: No exact linear relationship among X's
  6. No Endogeneity: Errors uncorrelated with X's
‚ö†Ô∏è Important

Violations don't invalidate the model ‚Äî they affect reliability of tests and inferences.

Detecting Violations

Problem Detection
Heteroskedasticity Breusch-Pagan test
Serial correlation Durbin-Watson stat
Multicollinearity VIF > 10

üìä ANOVA & Fit

Sum of Squares
$$SST = SSR + SSE$$
Total = Explained + Unexplained
Measure Formula Meaning
SST $\sum(Y_i - \bar{Y})^2$ Total variation
SSR $\sum(\hat{Y}_i - \bar{Y})^2$ Explained
SSE $\sum(Y_i - \hat{Y}_i)^2$ Residuals

R¬≤ and Adjusted R¬≤

R-Squared
$$R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$$
% of variance explained by model
Adjusted R¬≤
$$R^2_{adj} = 1 - \left[\frac{n-1}{n-k-1}\right](1 - R^2)$$
Penalizes adding weak variables
Use Adjusted R¬≤ when comparing models with different numbers of variables.

üß™ Hypothesis Testing

Individual Coefficients (t-test)

t-Statistic
$$t = \frac{\hat{\beta}_j - \beta_{j,0}}{SE(\hat{\beta}_j)}$$
df = n - k - 1
Test H‚ÇÄ Decision
Two-tailed Œ≤ = 0 Reject if |t| > tcrit
One-tailed Œ≤ ‚â§ 0 Reject if t > tcrit

Overall Model (F-test)

F-Statistic
$$F = \frac{MSR}{MSE} = \frac{SSR/k}{SSE/(n-k-1)}$$
F-test answers: "Are any of the X variables useful?" If F > Fcrit, at least one Œ≤ ‚â† 0.

Prediction

üìä Example

Model: EPS = 0.50 + 0.03(Rev) + 0.02(Assets)

If Rev = 100, Assets = 50:

EPS = 0.50 + 3 + 1 = 4.50
