Coefficient of Determination (R¬≤)

R¬≤ measures the proportion of total variation in the dependent variable that is explained by the independent variable(s) in the regression model.

R-Squared Formula
\[ R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST} \]
SSR = Sum of Squares Regression | SSE = Sum of Squares Error | SST = Total Sum of Squares

üìä R¬≤ Interpretation

‚Ä¢ R¬≤ = 0.70 ‚Üí 70% of Y's variation explained by X

‚Ä¢ Higher R¬≤ = Better fit

‚Ä¢ Range: 0 to 1 (or 0% to 100%)

‚ö†Ô∏è R¬≤ Limitation

‚Ä¢ Always increases when adding variables

‚Ä¢ Even if new variable is irrelevant!

‚Ä¢ Can lead to overfitting

Adjusted R¬≤

Adjusted R¬≤ penalizes adding variables that don't improve the model. It only increases if the new variable improves fit more than expected by chance.

Adjusted R-Squared
\[ \bar{R}^2 = 1 - \left(\frac{n-1}{n-k-1}\right)(1 - R^2) \]
n = sample size | k = number of independent variables

R¬≤ vs Adjusted R¬≤: When comparing models with different numbers of variables, always use Adjusted R¬≤. R¬≤ will always increase (or stay same) when adding variables, even useless ones!

üìå Key Insight: |t| > 1 Rule

If added variable's |t-statistic| > 1.0 ‚Üí Adjusted R¬≤ increases

If added variable's |t-statistic| < 1.0 ‚Üí Adjusted R¬≤ decreases

Warning: |t| = 1.0 is NOT significance! Significance needs |t| ‚âà 2.0

AIC & BIC: Model Selection Criteria

When adjusted R¬≤ is not enough, use information criteria to select the best model from a set of candidates.

Akaike's Information Criterion (AIC)
\[ AIC = n \times \ln\left(\frac{SSE}{n}\right) + 2(k + 1) \]
n = observations | k = independent variables | 2(k+1) = complexity penalty | LOWER = BETTER
Bayesian Information Criterion (BIC)
\[ BIC = n \times \ln\left(\frac{SSE}{n}\right) + \ln(n) \times (k + 1) \]
Heavier penalty: ln(n)(k+1) since ln(n) > 2 for n > 7 | LOWER = BETTER

üéØ AIC

‚Ä¢ Lighter penalty: 2(k+1)

‚Ä¢ Preferred for PREDICTION

‚Ä¢ Tends to select larger models

‚Ä¢ Memory: AIC for Accurate Prediction

‚öñÔ∏è BIC

‚Ä¢ Heavier penalty: ln(n)(k+1)

‚Ä¢ Preferred for GOODNESS OF FIT

‚Ä¢ Tends to select smaller models

‚Ä¢ Memory: BIC for Best fit / Basic model

AIC/BIC Values Are Meaningless Alone! Only relative values among models matter. You can ONLY compare models with the same dependent variable.

üìù Model Selection Example

Given 5 candidate models for portfolio returns:

Model R¬≤ Adj R¬≤ AIC BIC
Factor 1 only 0.541 0.531 19.08 22.90
Factors 1-2 0.541 0.521 21.08 26.81
Factors 1-4 0.615 0.580 16.33 25.89

‚Ä¢ For Prediction: Choose 4-factor model (lowest AIC = 16.33)

‚Ä¢ For Parsimony: Choose 1-factor model (lowest BIC = 22.90)

F-Test for Overall Significance

F-Statistic
\[ F = \frac{MSR}{MSE} = \frac{SSR/k}{SSE/(n-k-1)} \]
Degrees of freedom: df‚ÇÅ = k, df‚ÇÇ = n - k - 1

H‚ÇÄ: Null Hypothesis

All slope coefficients = 0

\( b_1 = b_2 = ... = b_k = 0 \)

Model has no explanatory power

H‚ÇÅ: Alternative Hypothesis

At least one slope ‚â† 0

Model explains some variation

Reject H‚ÇÄ if F > F-critical

F-Test vs t-Test: A significant F-test means the model as a whole is significant, but it does NOT mean all individual coefficients are significant. You still need t-tests for individual variables!

Slope Coefficient Interpretation

For a regression: \( Y = b_0 + b_1X_1 + b_2X_2 + ... + b_kX_k \)

Coefficient Meaning
\[ b_i = \frac{\Delta Y}{\Delta X_i} \text{ (holding all other X's constant)} \]
A one-unit increase in X·µ¢ ‚Üí b·µ¢ unit change in Y, ceteris paribus

t-Test for Individual Coefficients

t-Statistic
\[ t = \frac{b_i - 0}{s_{b_i}} = \frac{b_i}{s_{b_i}} \]
H‚ÇÄ: b·µ¢ = 0 (variable has no effect) | df = n - k - 1
üìù Example: Interpreting Output

Regression output shows: Sales = 50 + 2.5(Advertising) + 0.8(Price)

‚Ä¢ b‚ÇÅ = 2.5: Each $1 increase in Advertising ‚Üí $2.50 increase in Sales (holding Price constant)

‚Ä¢ b‚ÇÇ = 0.8: Each $1 increase in Price ‚Üí $0.80 increase in Sales (holding Advertising constant)

Standard Error of Estimate (SEE)

Standard Error of Estimate
\[ SEE = \sqrt{MSE} = \sqrt{\frac{SSE}{n-k-1}} \]
Measures the standard deviation of residuals (prediction errors)
‚ö†Ô∏è Interpretation

‚Ä¢ Lower SEE = Better model fit (smaller prediction errors)

‚Ä¢ SEE is in the same units as Y

‚Ä¢ Useful for comparing models with same dependent variable

üìè Prediction vs Confidence Intervals

üí° LOS: Learning Objective

‚Ä¢ Compare and contrast prediction intervals and confidence intervals

Two Types of Intervals

üéØ Confidence Interval

For the mean value of Y

Given X = x‚ÇÄ, what is Œº·µß?

Narrower interval

Less uncertainty about the average

üìä Prediction Interval

For an individual value of Y

Given X = x‚ÇÄ, what will Y be?

Wider interval

More uncertainty about any single observation

Key Relationship
Prediction Interval > Confidence Interval
Prediction intervals include extra variance from individual observation uncertainty

Common Exam Question: "Which interval is wider?" ‚Äî Prediction interval is ALWAYS wider because it accounts for both the uncertainty in the mean AND the variability of individual observations around that mean.

üí° LOS: Learning Objective

‚Ä¢ Evaluate regression assumptions using diagnostic plots

Key Assumptions to Check

  • Linearity: Relationship between X and Y is linear
  • Homoskedasticity: Constant variance of residuals
  • Independence: No autocorrelation in residuals
  • Normality: Residuals are normally distributed

‚ö†Ô∏è What to do if assumptions violated?

‚Ä¢ Heteroskedasticity: Use robust standard errors or transform variables

‚Ä¢ Autocorrelation: Use Newey-West standard errors or time-series models

‚Ä¢ Non-normality: For large samples, CLT may apply; otherwise transform Y

üéØ Exam Summary

Quick Formulas

Measure Formula When to Use
R¬≤ \( 1 - \frac{SSE}{SST} \) Single model evaluation
Adjusted R¬≤ \( 1 - \frac{n-1}{n-k-1}(1-R^2) \) Comparing models with different # of variables
F-stat \( \frac{MSR}{MSE} \) Overall model significance
t-stat \( \frac{b_i}{s_{b_i}} \) Individual coefficient significance
SEE \( \sqrt{MSE} \) Prediction accuracy
AIC \( n \ln(SSE/n) + 2(k+1) \) Model selection for prediction
BIC \( n \ln(SSE/n) + \ln(n)(k+1) \) Model selection for parsimony

‚ö†Ô∏è Exam Traps

1. R¬≤ Always Increases: Adding any variable (even random!) increases R¬≤. Use Adjusted R¬≤ for model comparison.

2. Significant F ‚â† All Significant Coefficients: F-test is for the whole model; individual coefficients need separate t-tests.

3. Prediction vs Confidence: Prediction intervals are WIDER because they include individual observation variance.

4. Degrees of Freedom: For F-test: df‚ÇÅ = k, df‚ÇÇ = n - k - 1 (not n - k or n - 1)

5. |t| > 1 ‚â† Significance: |t-stat| > 1.0 increases Adjusted R¬≤, but significance requires |t| ‚âà 2.0. Don't confuse these thresholds!

6. AIC/BIC Values Alone Mean Nothing: Only relative values matter. Can ONLY compare models with the same dependent variable.

7. AIC vs BIC: AIC for Prediction (larger models), BIC for Parsimony (smaller models). BIC penalty is heavier since ln(n) > 2.
