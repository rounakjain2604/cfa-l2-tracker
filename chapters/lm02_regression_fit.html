P.1 The Logic of Hypothesis Testing

Plain English: Hypothesis testing asks: "Is this result real, or could it have happened by chance?"

‚öñÔ∏è The Courtroom Analogy (CRITICAL)

‚Ä¢ Defendant is innocent until proven guilty = H‚ÇÄ is assumed true until rejected

‚Ä¢ Prosecution must prove guilt beyond reasonable doubt = We need strong evidence to reject H‚ÇÄ

‚Ä¢ "Not guilty" ‚â† "innocent" = "Fail to reject H‚ÇÄ" ‚â† "H‚ÇÄ is true"

Hypothesis Symbol Meaning
Null H‚ÇÄ "Nothing is happening" / "No effect" ‚Äî Assumed TRUE until rejected
Alternative H‚Çê "Something IS happening" ‚Äî What we're trying to prove

The Two Types of Errors

Error What Happens Memory Trick
Type I Reject H‚ÇÄ when it's TRUE "False Positive" ‚Äî You see something that isn't there
Type II Fail to reject H‚ÇÄ when it's FALSE "False Negative" ‚Äî You miss something real
Œ± (alpha) = significance level = probability of Type I error. Common values: 5% or 1%.

P.2 Understanding t-Tests

Plain English: A t-test asks: "Is this ONE coefficient significantly different from zero (or some value)?"

t-Statistic Formula
$$t = \frac{\hat{\beta} - \beta_0}{SE(\hat{\beta})} = \frac{\text{Estimated} - \text{Hypothesized}}{\text{Standard Error}}$$
üì° Signal-to-Noise Intuition

Think of t as a signal-to-noise ratio:

‚Ä¢ Large |t| ‚Üí Strong signal relative to noise ‚Üí Likely significant

‚Ä¢ Small |t| ‚Üí Weak signal drowned by noise ‚Üí Probably just chance

Rule of Thumb: |t| > 2 usually means significant at 5%

Why? Critical t-value at 5% significance ‚âà 1.96 ‚âà 2 for large samples.

üß† STEP 1: INTUITION FIRST

Why Does This Concept Exist?

Imagine you're a portfolio manager at an Indian AMC trying to explain what drives your fund's returns. You run a regression with 5 factors (say, market, size, value, momentum, quality). The model spits out numbers‚Äîbut how do you know if these factors actually explain the returns, or if you've just added noise?

The Core Problem:

  • In simple regression (1 independent variable), R¬≤ tells you what percentage of the dependent variable's variance is explained by the model.
  • But in multiple regression, every time you add a variable, R¬≤ will never decrease‚Äîeven if that variable is useless garbage.
  • This creates a dangerous trap: you could keep adding variables and R¬≤ keeps rising, fooling you into thinking you have a "better" model when you're actually overfitting.
üçõ Real-World Analogy (Indian Context)

Think of it like adding masalas to a dish. Adding jeera and dhania improves taste (genuine improvement). But if you keep adding random spices just because "more spices = better," the dish becomes a confused mess. You need a metric that penalizes unnecessary additions.

What This LOS Solves:

  • Adjusted R¬≤ adjusts for the number of variables, penalizing model complexity
  • AIC and BIC provide rigorous model comparison frameworks
  • We learn when to add variables and when to stop

üìö STEP 2: CFAI CONCEPT BREAKDOWN

2.1 Coefficient of Determination (R¬≤) in Multiple Regression

CFAI Definition: R¬≤ is the ratio of the variation of the dependent variable explained by the independent variables (Sum of Squares Regression, SSR) to the total variation of the dependent variable (Sum of Squares Total, SST).
R-Squared Formula
$$R^2 = \frac{SSR}{SST} = \frac{\sum(\hat{Y}_i - \bar{Y})^2}{\sum(Y_i - \bar{Y})^2}$$
Alternative Expression
$$R^2 = 1 - \frac{SSE}{SST}$$

Variable Definitions:

  • n = Number of observations
  • Y·µ¢ = Actual observed value of dependent variable
  • ≈∂·µ¢ = Predicted value from regression model
  • »≤ = Mean of the dependent variable
  • SSR = Sum of Squares Regression (explained variation)
  • SSE = Sum of Squares Error/Residual (unexplained variation)
  • SST = Sum of Squares Total = SSR + SSE

Interpretation: R¬≤ = 0.62 means 62% of the variation in Y is explained by the independent variables in the model.

2.2 The Fatal Flaw of R¬≤ in Multiple Regression

CFAI explicitly states three problems with R¬≤ in multiple regression:

Problem Explanation Exam Trap
1. Cannot tell if coefficients are significant High R¬≤ doesn't mean Œ≤ coefficients are statistically ‚â† zero Don't confuse model fit with coefficient significance
2. Cannot detect biased coefficients Model can have high R¬≤ but produce biased forecasts Residual analysis still required
3. Cannot distinguish good vs bad models Good models may have low R¬≤; overfitted models have high R¬≤ High R¬≤ ‚â† Good Model
Overfitting Defined (CFAI)

"Overfitting is a situation in which the model is too complex, meaning there may be too many independent variables relative to the number of observations. A result is that the coefficients may not represent true relationships with the dependent variable."

When you see a model with many variables and high R¬≤, immediately suspect overfitting. CFAI loves testing this concept.

2.3 Adjusted R¬≤ (RÃÑ¬≤)

CFAI Definition: "An alternative measure of goodness of fit that does not automatically increase when another independent variable is added. It adjusts for degrees of freedom by incorporating the number of independent variables."
Adjusted R-Squared (Primary Form)
$$\bar{R}^2 = 1 - \left[\frac{n - 1}{n - k - 1}\right] \times (1 - R^2)$$
n = observations | k = independent variables (excluding intercept)

2.4 Key Properties of Adjusted R¬≤

Property Explanation
Always ‚â§ R¬≤ When k ‚â• 1, adjusted R¬≤ is strictly less than R¬≤
Can be negative Unlike R¬≤ (minimum = 0), adjusted R¬≤ can go below zero
Increases with "good" variables If added variable's |t-statistic| > 1.0
Decreases with "bad" variables If added variable's |t-statistic| < 1.0
Critical Insight (Exam Favorite):

The threshold of |t-statistic| = 1.0 is NOT the same as statistical significance!

  • At 5% significance with ~50 observations, critical t ‚âà 2.0
  • So a variable with |t| = 1.2 will increase adjusted R¬≤ but is still not statistically significant
  • "Adjusted R¬≤ does not set a very high bar for the statistic to increase" ‚Äî CFAI
Exam Checkpoint ‚úì

Q: If adding a variable increases adjusted R¬≤, is that variable definitely significant?

A: NO! The bar for increasing adjusted R¬≤ (|t| > 1.0) is lower than typical significance thresholds (|t| > ~2.0).

2.5 Akaike's Information Criterion (AIC)

CFAI Definition: "A measure to evaluate a collection of models that explain the same dependent variable. AIC is a measure of model parsimony, so a lower AIC indicates a better-fitting model."
AIC Formula
$$AIC = n \times \ln\left(\frac{SSE}{n}\right) + 2(k + 1)$$
2(k+1) = Penalty term for complexity | LOWER = BETTER | Preferred for PREDICTION

2.6 Schwarz's Bayesian Information Criterion (BIC/SBC)

BIC Formula
$$BIC = n \times \ln\left(\frac{SSE}{n}\right) + \ln(n) \times (k + 1)$$
ln(n)(k+1) = Heavier penalty | LOWER = BETTER | Preferred for PARSIMONY

Key Points:

  • BIC applies a heavier penalty than AIC because ln(n) > 2 for n > 7
  • BIC tends to select smaller, simpler models than AIC

2.7 AIC vs. BIC Decision Framework

üéØ AIC

‚Ä¢ Penalty: 2(k+1) ‚Äî Lighter

‚Ä¢ Preferred for PREDICTION

‚Ä¢ Tends to select larger models

‚Ä¢ Memory: AIC = Accurate Prediction

‚öñÔ∏è BIC

‚Ä¢ Penalty: ln(n)(k+1) ‚Äî Heavier

‚Ä¢ Preferred for GOODNESS OF FIT

‚Ä¢ Tends to select smaller models

‚Ä¢ Memory: BIC = Best fit / Basic

Critical Rule: "The value of these measures considered alone is meaningless; the relative values of AIC or BIC among a set of models is what really matters." You can ONLY compare models with the same dependent variable.

üîó LOS 2.03: Joint Hypothesis Testing

üéØ Learning Outcome

"Formulate hypotheses on the significance of two or more coefficients in a multiple regression model and interpret the results of the joint hypothesis tests."

Why Not Just Use Individual t-Tests?

In multiple regression with correlated variables, there is often sharing of explanatory power between variables. Individual t-tests don't account for this correlation.

üèè Cricket Analogy

Think of Rohit and Dhawan as openers. Individually, neither might seem like the "best" opener statistically. But together as a pair, they've been devastating. You need a way to evaluate them as a joint unit, not just individually.

CFAI: "Often in multiple regression involving financial variables, there is some degree of correlation between the variables, so there may be some sharing of explanatory power that is not considered with the tests of individual slopes."

Unrestricted vs. Restricted Models

Model Type Definition Example
Unrestricted Full model with all variables Y = b‚ÇÄ + b‚ÇÅX‚ÇÅ + b‚ÇÇX‚ÇÇ + b‚ÇÉX‚ÇÉ + b‚ÇÑX‚ÇÑ + b‚ÇÖX‚ÇÖ + Œµ
Restricted Model with some slopes = 0 Y = b‚ÇÄ + b‚ÇÅX‚ÇÅ + b‚ÇÇX‚ÇÇ + b‚ÇÉX‚ÇÉ + Œµ (b‚ÇÑ = b‚ÇÖ = 0)
Nested Restricted is "nested" within unrestricted One model is special case of other

The Joint F-Test Formula

Joint F-Test for Nested Models
$$F = \frac{(SSE_{restricted} - SSE_{unrestricted}) / q}{SSE_{unrestricted} / (n - k - 1)}$$
q = number of restrictions | df‚ÇÅ = q | df‚ÇÇ = n ‚àí k ‚àí 1

Hypothesis Structure:

H‚ÇÄ: b‚ÇÑ = b‚ÇÖ = ... = 0 (all restricted slopes = 0)

H‚Çê: At least one of the restricted slopes ‚â† 0

Decision Rule:

Reject H‚ÇÄ if F-calculated > F-critical

F-test is always one-tailed (right side)

Logic Behind the Test:
  • SSE measures unexplained variation
  • If restricted model has much higher SSE, omitted variables were important
  • If SSE barely increases when variables are dropped, those variables weren't contributing

üìù Joint F-Test Calculation (1% significance)

H‚ÇÄ: b‚ÇÑ = b‚ÇÖ = 0 | q = 2 restrictions

$$F = \frac{(64.5176 - 56.6182)/2}{56.6182/44} = \frac{7.8994/2}{1.2868} = \frac{3.9497}{1.2868} = 3.07$$

Conclusion: Factors 4 and 5 do NOT jointly add significant explanatory power. Use the simpler 3-factor model.

Overall F-Test (Goodness of Fit)

Special case: test whether ALL slope coefficients = 0

Overall F-Statistic
$$F = \frac{MSR}{MSE} = \frac{SSR/k}{SSE/(n-k-1)}$$
df‚ÇÅ = k | df‚ÇÇ = n ‚àí k ‚àí 1

H‚ÇÄ: All Slopes = 0

b‚ÇÅ = b‚ÇÇ = ... = b‚Çñ = 0

Model has no explanatory power

H‚Çê: At Least One ‚â† 0

At least one b‚±º ‚â† 0

Model explains some variation

üìù Exhibit 6: Overall F-Test (5% significance)

From ANOVA: MSR = 18.1247, MSE = 1.2868

$$F = \frac{18.1247}{1.2868} = 14.09$$

Critical F(5, 44) at 5% = 2.427

Decision: 14.09 >> 2.427 ‚Üí Reject H‚ÇÄ

Conclusion: At least one factor significantly explains portfolio returns.

F-Test vs t-Test: A significant overall F-test means the model as a whole is significant, but it does NOT mean all individual coefficients are significant. You still need t-tests for individual variables!
