üìà R¬≤ and Adjusted R¬≤

Coefficient of Determination (R¬≤)

R¬≤ measures the proportion of total variation in Y that is explained by the independent variables.

R-Squared Formula
$$R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$$
SSR = Explained | SSE = Unexplained | SST = Total = SSR + SSE
3 Fatal Flaws of R¬≤ in Multiple Regression:
  1. Cannot tell if coefficients are statistically significant
  2. Cannot detect biased coefficients or predictions
  3. Cannot distinguish good vs bad models (high R¬≤ may = overfitting)

Adjusted R¬≤ ‚Äî The Fix

Adjusted R¬≤ penalizes adding variables that don't improve the model. It only increases if the new variable adds genuine explanatory power.

Adjusted R-Squared
$$\bar{R}^2 = 1 - \left[\frac{n-1}{n-k-1}\right] \times (1 - R^2)$$
n = observations | k = independent variables (excl. intercept)
Property R¬≤ Adjusted R¬≤
Minimum value 0 Can be negative
When variable added Always ‚â• May ‚Üë or ‚Üì
Threshold for ‚Üë Any variable |t| > 1.0
Penalizes complexity? No Yes
The |t| > 1 Rule (CRITICAL FOR EXAM):
  • If added variable's |t-statistic| > 1.0 ‚Üí Adjusted R¬≤ increases
  • If added variable's |t-statistic| < 1.0 ‚Üí Adjusted R¬≤ decreases

‚ö†Ô∏è Warning: |t| = 1.0 is NOT statistical significance! Significance needs |t| ‚âà 2.0

üìù Example: CAPEX Model

Model 1: ROA ~ CAPEX only ‚Üí R¬≤ = 0.880, Adj R¬≤ = 0.875

Model 2: ROA ~ CAPEX + ADV ‚Üí R¬≤ = 0.881, Adj R¬≤ = 0.870

ADV coefficient: t-stat = ‚àí0.332

Analysis:

  • R¬≤ increased (0.880 ‚Üí 0.881) ‚Äî expected, R¬≤ never decreases
  • Adj R¬≤ decreased (0.875 ‚Üí 0.870)
  • |t-stat| = 0.332 < 1.0 ‚Üí Variable hurts model

Conclusion: Do NOT add ADV. Stick with Model 1.

‚öñÔ∏è AIC & BIC Model Selection

When adjusted R¬≤ is not enough, use information criteria to select the best model from a set of candidates.

Akaike's Information Criterion (AIC)
$$AIC = n \times \ln\left(\frac{SSE}{n}\right) + 2(k + 1)$$
Penalty: 2(k+1) ‚Äî Lighter | LOWER = BETTER | Preferred for PREDICTION
Bayesian Information Criterion (BIC)
$$BIC = n \times \ln\left(\frac{SSE}{n}\right) + \ln(n) \times (k + 1)$$
Penalty: ln(n)(k+1) ‚Äî Heavier | LOWER = BETTER | Preferred for PARSIMONY

üéØ AIC

‚Ä¢ Lighter penalty: 2(k+1)

‚Ä¢ Best for PREDICTION

‚Ä¢ Tends to select larger models

‚Ä¢ Memory: AIC = Accurate Prediction

‚öñÔ∏è BIC

‚Ä¢ Heavier penalty: ln(n)(k+1)

‚Ä¢ Best for GOODNESS OF FIT

‚Ä¢ Tends to select smaller models

‚Ä¢ Memory: BIC = Best fit / Basic

AIC/BIC Values Alone Are Meaningless!

Only relative values matter. You can ONLY compare models with the same dependent variable.

F-Test for Overall Significance

F-Statistic
$$F = \frac{MSR}{MSE} = \frac{SSR/k}{SSE/(n-k-1)}$$
df‚ÇÅ = k (numerator) | df‚ÇÇ = n ‚àí k ‚àí 1 (denominator)

H‚ÇÄ: Null Hypothesis

All slope coefficients = 0

$b_1 = b_2 = ... = b_k = 0$

Model has no explanatory power

H‚ÇÅ: Alternative

At least one slope ‚â† 0

Model explains some variation

Reject H‚ÇÄ if F > F-critical

F-Test vs t-Test: A significant F-test means the model as a whole is significant, but it does NOT mean all individual coefficients are significant. You still need t-tests for individual variables!

üßÆ Interpreting Coefficients

Slope Coefficient Meaning

Coefficient Interpretation
$$b_i = \frac{\Delta Y}{\Delta X_i}$$ (holding all other X's constant)
A one-unit increase in X·µ¢ ‚Üí b·µ¢ unit change in Y, ceteris paribus
THE GOLDEN RULE: You must always include the phrase "holding all other variables constant" when interpreting a coefficient.

t-Test for Individual Coefficients

t-Statistic
$$t = \frac{b_i - 0}{s_{b_i}} = \frac{b_i}{s_{b_i}}$$
H‚ÇÄ: b·µ¢ = 0 (variable has no effect) | df = n ‚àí k ‚àí 1
üìù Example: Interpreting Output

Regression: Sales = 50 + 2.5(Advertising) + 0.8(Price)

  • b‚ÇÅ = 2.5: Each $1 ‚Üë in Advertising ‚Üí $2.50 ‚Üë in Sales (holding Price constant)
  • b‚ÇÇ = 0.8: Each $1 ‚Üë in Price ‚Üí $0.80 ‚Üë in Sales (holding Advertising constant)

Standard Error of Estimate (SEE)

SEE Formula
$$SEE = \sqrt{MSE} = \sqrt{\frac{SSE}{n-k-1}}$$
Measures standard deviation of residuals | Lower = Better fit | Same units as Y

üìè Prediction vs Confidence Intervals

üéØ Confidence Interval

For the mean value of Y

Given X = x‚ÇÄ, what is Œº·µß?

Narrower interval

Less uncertainty about average

üìä Prediction Interval

For an individual value of Y

Given X = x‚ÇÄ, what will Y be?

Wider interval

More uncertainty about single obs

Prediction Interval > Confidence Interval ‚Äî ALWAYS

Prediction intervals include extra variance from individual observation uncertainty around the mean.

Common Exam Question: "Which interval is wider?"

Answer: Prediction interval is ALWAYS wider because it accounts for both:

  • Uncertainty in the mean (what CI captures)
  • Variability of individual observations around that mean

‚ö†Ô∏è The 7 Exam Traps

1. R¬≤ Always Increases: Adding any variable (even random noise!) increases R¬≤. Use Adjusted R¬≤ for model comparison.
2. Significant F ‚â† All Significant Coefficients: F-test is for the whole model; individual coefficients need separate t-tests.
3. Prediction vs Confidence: Prediction intervals are WIDER because they include individual observation variance.
4. Degrees of Freedom: For F-test: df‚ÇÅ = k, df‚ÇÇ = n ‚àí k ‚àí 1 (not n‚àík or n‚àí1)
5. |t| > 1 ‚â† Significance: |t-stat| > 1.0 increases Adjusted R¬≤, but statistical significance requires |t| ‚âà 2.0. Don't confuse these!
6. AIC/BIC Values Alone Mean Nothing: Only relative values matter. Can ONLY compare models with same dependent variable.
7. AIC vs BIC: AIC for Prediction (larger models). BIC for Parsimony (smaller models). BIC penalty is heavier since ln(n) > 2.
